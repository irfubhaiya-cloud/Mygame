# 🎰 Lucky 18 Casino Betting Game

Welcome to the **Lucky 18 Casino Betting Game**! This is a simple, interactive, and fully responsive web-based betting game built specifically for a casino-style web app.

## ✨ Features

* **Number Grid (1-18):** A clean and interactive grid for users to pick their lucky number.
* **Virtual Balance System:** Players start with a virtual balance of ₹1000.
* **10x Winning Multiplier:** If the randomly generated number matches the player's selected number, they win 10 times (10x) their bet amount!
* **Interactive UI:** Features a dark mode casino theme, button highlight effects, and a suspenseful "spinning" delay.
* **Responsive Design:** Works perfectly on both mobile webviews and desktop browsers.

## 🎮 How to Play

1. Open the game link.
2. Check your current balance at the top (Default: ₹1000).
3. Select any number from the **1 to 18** grid.
4. Enter your **Bet Amount** in the input box (must be greater than 0).
5. Click on the **Place Bet & Spin** button.
6. A random number will be generated. If it matches your selection, you win the jackpot!

## 🛠️ Technologies Used

* **HTML5:** For the core structure.
* **CSS3:** For styling, Flexbox/Grid layouts, and animations.
* **JavaScript (Vanilla):** For the game logic, random number generation, and balance calculation.
